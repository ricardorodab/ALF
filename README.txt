Información extra:

0. Para compilar se puede usar ant o cualquier programa equivalente de manejador de xml y correr el comando "ant compile".

1. Para correr el programa correctamente se necesita una versión de java mayor o igual a java8.0_20

2. Para más información sobre la aplicación puede contactar al correo ricardo_rodab@ciencias.unam.mx

3. Basta con ejecutar el archivo Proyecto01.class después de compilarlo para poder obtener toda la información.

4. En la carpeta "EjemploEntrada" viene algunos ejemplos de como el programa lee los autómatas.

5. Para conocer el formato exacto en que debe de estar la entrada correr el programa con "java Proyecto01 info".

6. Para ver toda la documentación sólo falta generarla con "ant doc".

7. La clase principal Proyecto01 se encuentra en el archivo build/ después de compilar.
